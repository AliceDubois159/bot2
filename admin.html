Administration des Plantes

Guide d'utilisation d'Airtable

Ce guide vous explique comment utiliser Airtable pour gérer votre catalogue de plantes.

Configuration initiale

1 Créez un compte Airtable: Rendez-vous sur airtable.com et créez un compte gratuit si vous n'en avez pas déjà un.
2 Créez une nouvelle base: Nommez-la, par exemple, "Catalogue de Plantes".
3 Créez une table "Plantes": Cette table contiendra toutes les informations sur vos plantes.

Structure de la table Plantes

Voici les champs à créer dans votre table Plantes:

  • Nom (Type: Texte simple)
  • Type (Type: Texte simple)
  • Catégorie (Type: Sélection simple) - Ajoutez toutes vos catégories
  • Provenance (Type: Sélection simple) - Ajoutez toutes vos provenances
  • Image (Type: Pièce jointe) - Pour télécharger des images
  • Prix Unité (Type: Texte simple)
  • Prix par 3 (Type: Texte simple)
  • Prix par 5 (Type: Texte simple)
  • Vidéos (Type: Texte long) - Entrez les URLs des vidéos, une par ligne
  • Description (Type: Texte long)

Astuce: Vous pouvez ajouter d'autres champs selon vos besoins.

Ajouter et gérer les plantes

1 Ajouter une plante: Cliquez sur le "+" au début d'une ligne pour ajouter une nouvelle plante.
2 Télécharger une image: Dans le champ Image, cliquez pour télécharger une image depuis votre téléphone ou ordinateur.
3 Ajouter des vidéos: Dans le champ Vidéos, entrez les URLs de vos vidéos, une par ligne.
4 Modifier une plante: Cliquez simplement sur n'importe quel champ pour le modifier.
5 Supprimer une plante: Cliquez sur les trois points à droite de la ligne et sélectionnez "Supprimer".

Astuce: Toutes les modifications sont sauvegardées automatiquement, et votre bot Telegram sera mis à jour automatiquement (le délai de mise à jour est configuré dans les paramètres du bot).

Obtenir les informations API

Pour que votre bot Telegram puisse se connecter à Airtable, vous aurez besoin des informations suivantes:

1 Trouver votre API Key: Allez dans votre compte Airtable, section "API", et générez une clé API si vous n'en avez pas.
2 Trouver votre Base ID: Ouvrez votre base, cliquez sur "Aide" dans le coin supérieur droit, puis "API Documentation". Votre Base ID se trouve en haut de la page.
3 Configurer votre fichier .env: Utilisez ces informations pour mettre à jour votre fichier .env avec les variables AIRTABLE_API_KEY, AIRTABLE_BASE_ID et AIRTABLE_TABLE_NAME.
