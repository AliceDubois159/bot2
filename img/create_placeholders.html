Générateur d'Images Placeholder

Créer des Images Placeholder

Utilisez cet outil pour générer des images placeholder pour votre catalogue de plantes.

Comment utiliser les images

1. Cliquez sur le bouton "Télécharger" sous chaque image que vous souhaitez utiliser.

2. Renommez les fichiers téléchargés en placeholder1.jpg, placeholder2.jpg, etc.

3. Placez les fichiers dans le dossier webapp/img/
